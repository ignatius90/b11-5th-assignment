There are a couple of ways to select an element in dom. They are, getElementByID(), getElementsByClassName(), getElementsByTagName(), querySelectorAll(), querrySelector().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML, innerText & textContent are the properties of Dom in html. innerHTML provide the the HTML markup and text content at the same time whereas innerText provide the content inside of an element and ignore html tags. And textContent provide the text content of elements & it contains white spaces, hidden elements, newlines etc.

Question-3: What is event delegation in the DOM?

event delegation is a process in Javascript where a single event listener is attached and will trigger all selector that will match.

Question-4: What is event bubbling in the DOM?

Event bubbling is a type of DOM where describe how the event propagate or bubble up when it triggered. The event stat at the target element and propagate up through its parent element to the root.
